# css2026-research-profile

This repository contains a simple Streamlit application showcasing my data science profile and work.

## Live Application
https://css2026-research-profile.streamlit.app

## About
The app presents:
- A brief professional profile
- Selected data science and analytics projects
- Technical skills
- Contact information

It was built to demonstrate the ability to create and deploy a Streamlit application on Streamlit Cloud.

## Technologies Used
- Python
- Streamlit

## How to Run Locally
```bash
pip install streamlit
streamlit run app.py
